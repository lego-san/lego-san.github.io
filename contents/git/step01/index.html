Git Branchコマンドの使い方

目次

ブランチ操作のヘルプ確認方法

git branch -h

ブランチの削除

git branch -d [ブランチ名]
git branch --delete [ブランチ名]

ブランチの強制削除

git branch -D [ブランチ名]

リモートレポジトリの追加

git remote add origin [レポジトリ名]

リモートレポジトリの削除

git remote rm origin

ユーザ情報確認方法

git config -l

ユーザ情報設定方法

git config --local user.name [name]
git config --local user.email [email]

ユーザ情報切り替え

git config user.name [name]
