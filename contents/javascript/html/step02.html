Java Scriptの外部参照

Java ScriptはHTML内部に直接書くこともできますが、外部に用意されているがいるファイルを読み込むことができます。

対象のJava Script

今回使用するのはHMLの要素を書き換えるこちらのJava Scriptです。


  <script type="text/javascript">
    document.getElementById("ID_STEP01").innerHTML = "Java Scriptのはじめの一歩";
  </script>

Java Scriptファイルの用意

まずJava Scriptファイルを用意します。ファイル名をgetElementById、拡張子を".js"としてます。 replace_comment(id, comment)関数の中で指定されたIDのタグに対して引数のコメントを挿入しています。

ファイル名：getElementById.js


  function replace_comment(id, comment){
    document.getElementById(id).innerHTML = comment;
  }

Java Scriptファイルで用意した関数の呼び出し

外部の>Java Scriptファイルを使うには、HEADタグで参照することを宣言する必要があります。


  <head>
    <script type="text/javascript" src="./getElementById.js"></script>
  </head>

HEADタグで宣言できたら本文中にこのように記載します。


  <script>
    replace_comment("ID_STEP01", "Java Scriptのはじめの一歩");
  </script>

ちょっとした解説

Java Script内でもHTMLの要素はgetElementById()関数でIDを指定し、 innerHTMLに値を入れることでコメントを挿入することができます。

ファイル名：getElementById.js


  function replace_comment(id, comment){
    document.getElementById(id).innerHTML = comment;
  }

HTML本文中では、Java Scriptで定義したreplace_comment関数をコールしてます。


  <script>
    replace_comment("ID_STEP01", "Java Scriptのはじめの一歩");
  </script>
