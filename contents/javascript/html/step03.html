事前準備

HTMLのFooterを置きたい場所に、id="ID_FOOTER"を割り当てたdiv領域を用意しておきます。


 <div id="ID_FOOTER"></div>

シンプルなFooter

シンプルなFooterはこのように記載できます。

ファイル名：footer.js


  function create_footer(){
    var contents = "FOOTERです。";
    element = document.getElementById("ID_FOOTER");
    var p = document.createElement('P');
    p.innerHTML = contents;
    element = element.appendChild(p);
  }

呼び出し元のHTMLはこのようになります。

ファイル名：main.html


  <div id="ID_FOOTER">
    <script>
      create_footer();
    </script>
  </div>

ちょっとした改善：IDを可変にする

シンプルなFooterはこのように記載できます。

ファイル名：footer.js


  function create_footer(id){
    var contents = "FOOTERです。";
    element = document.getElementById(id);
    var p = document.createElement('P');
    p.innerHTML = contents;
    element = element.appendChild(p);
  }

呼び出し元のHTMLはこのようになります。

ファイル名：main.html


  <div id="ID_FOOTER">
    <script>
      create_footer("ID_FOOTER");
    </script>
  </div>

ちょっとした改善：classを指定する

classを指定すれば、Styleシートで色やサイズ、位置などを簡単にすることができます。

ファイル名：footer.js


  function create_footer(id){
    var contents = "FOOTERです。";
    element = document.getElementById(id);
    var p = document.createElement('P');
    p.innerHTML = contents;
    p.classList.toggle("my_class");
    element = element.appendChild(p);
  }

ちょっとした改善：ページ読み込み時に起動する

Java Scriptでwindow.onloadを設定しておけば、ページ読み込み時に指定関数を呼び出してくれます。

ファイル名：footer.js


  window.onload = function(){
    create_footer("ID_FOOTER");
  }

呼び出し元のHTMLはこのようになります。

ファイル名：main.html


  <div id="ID_FOOTER"></div>
