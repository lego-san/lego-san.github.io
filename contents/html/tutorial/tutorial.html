STEP1 とにかく書いてみる!

まずはこちらをコピー&ペーストするか、 こちら からダウンロードして手元にファイルを準備してください。 コピー&ペーストでファイルを用意する方は、 ファイル名を「サンプル.html」などのようにして、ファイル拡張子をhtmlとしてデスクトップなどに保存してください。

最低限の記述で最初のページを書いてみます。

サンプル.html

<!DOCTYPE html>
<html lang="ja" dir="ltr">
  <head>
    <meta charset="utf-8">
    <title>レゴで遊んで学ぶ</title>
  </head>
  <body>
    <p>レゴで遊んだ写真を掲載します。</p>
  </body>
</html>

STEP4 ルール1 タグを使って文章を書く

まずはシンプルに考えるために赤字部分だけを確認します。

タグルールの確認

<!DOCTYPE html>
<html lang="ja" dir="ltr">
  <head>
    <meta charset="utf-8">
    <title>レゴで遊んで学ぶ</title>
  </head>
  <body>
    <p>レゴで遊んだ写真を掲載します。</p>
  </body>
</html>

この部分だけ見てみると、<p>と</p> の間に"レゴで遊んだ写真を掲載します。"というコメントの記載があります。 この"<"と">"で囲まれた部分がタグです。 このサンプルの場合、<p>のほかに<title>、<head>、<body>などもタグです。 なお、今回ピックアップした<p>はparagraph(段落、節)のpです。文章を書くときに使うタグです。

paragraphタグの使用例


　<p>レゴで遊んだ写真を掲載します。</p>

タグは開始タグと終了タグの2種類あります。 今回の場合、開始タグが<p>で、 終了タグがバックスラッシュ" / "が付いた</p>というルールがあります。 文章はこの開始タグ<p>と終了タグ</p>の間に書く必要があります。

他のタグも同様に、例えば、ウェブページタイトルはtitleの開始タグ<title>と終了タグ</title>の間に、 HTMLヘッダと呼ばれているものはheadの開始タグ<head>と終了タグ</head>の間に書くといった 開始タグと終了タグの間に記述するというルールがあります。

STEP5 ルール2 headとbodyを使ってHTMLを構成する

まずはhead(ヘッダ)を確認してみます。

HEADの確認

<!DOCTYPE html>
<html lang="ja" dir="ltr">
  <head>
    <meta charset="utf-8">
    <title>レゴで遊んで学ぶ</title>
  </head>
  <body>
    <p>レゴで遊んだ写真を掲載します。</p>
  </body>
</html>

headにはウェブページの情報を記述します。 開始タグは<head>、終了タグは</head>です。 その間に<meta charset="utf-8">や<title>xxx</title>がありますが、 これがheadに記載されたウェブページの情報となります。

<meta charset="utf-8">は文字形式(キャラクターセット)を示す情報ですが、まずはこのまま記載しておいてください。 プログラミングに親しい人ならよくご存じだと思いますし、そうでない人は深入りする必要はありません。

<title>xxx</title>はその名の通りタイトルです。ウェブページのタイトルを記載しておきます。 ただしここの情報はディスプレイには表示されません。Googleなどで検索したときに検索ブラウザ上で表示されるタイトルになります。 (ウェブブラウザのタブには表示されているかもしれません。)


つぎにbody(ボディ)を確認してみます。

BODYの確認

<!DOCTYPE html>
<html lang="ja" dir="ltr">
<head>
  <meta charset="utf-8">
  <title>レゴで遊んで学ぶ</title>
</head>
<body>
  <p>レゴで遊んだ写真を掲載します。</p>
</body>
</html>

bodyにはウェブページ本文を記述します。 開始タグは<body>、終了タグは</body>です。 その間に<p>レゴで遊んだ写真を掲載します。</p>がありますが、 これがbodyに記載されたウェブページ本文の情報となります。

今回は<p>タグがひとつですが、必要に応じて複数記載することができます。 タイトルタグと異なり、これはウェブページ上に記載される情報となります。

STEP6 その他のルール

HTMLを書くときのお作法があります。1つ目はDocument Type(ドキュメントタイプ)です。

ドキュメントタイプの確認

<!DOCTYPE html>
<html lang="ja" dir="ltr">
  <head>
    <meta charset="utf-8">
    <title>レゴで遊んで学ぶ</title>
  </head>
  <body>
    <p>レゴで遊んだ写真を掲載します。</p>
  </body>
</html>

これはドキュメントタイプを宣言するタグです。これによりHTMLドキュメントであることを明示しています。これはルールに従って書いておきましょう。

DOCTYPEの記載方法


　<!DOCTYPE html>

2つ目はhtmlタグです。DOCTYPE以外の全てのタグを囲むことになります。こちらも必ず書く必要があります。

HTMLタグの確認

<!DOCTYPE html>
<html lang="ja" dir="ltr">

  </head>
  <body>
    <p>レゴで遊んだ写真を掲載します。</p>
  </body>
</html>
