【レゴ研鑽】HTML 入力フォームの作り方

ウェブページでよく見かける入力フォーム。HTMLではどのように表現するのでしょうか。 今回はbuttonタグ、inputタグを紹介します。

目次

ウェブページで見かける入力フォームの例

今回紹介するのはこちらの入力フォームです。サンプルなので実際使いたいものとは異なるかもしれませんがそこはアレンジしてください。

この入力フォームはテキストコピーボタンを押すと、上部の入力ボックスの内容が下部の入力ボックスにコピーされます。 そして、リセットボタンを押すと初期状態にもどります。 ボタンはbuttonタグで、入力ボックスはinputタグで作成しました。

サンプルフォーム



サンプルフォームHTML

<form>
  <input type="text" name="textbox1" value="ここの値をコピーします。"><br>
  <input type="text" name="textbox2"><br>
  <button type="button" onclick="textbox2.value=textbox1.value;">テキストコピー</button>
  <button type="reset">リセット</button>
</form>

Inputタグ

Inputタグを使うと入力ボックスを作ることができます。 typeには入力形式を入れておきます。これは種類がたくさんありますので、今回はtextタイプのみを説明します。 value属性に文字を入れておけば初期状態としてその文字を表示することができます。入れておくことも可能ですし、 何も入れておかないこともできます。

Inputサンプル

</form>

Buttonタグ

Buttonタグを使うとボタンを作ることができます。 typeの種類はbutton/reset/submitの3種類ありますが、今回は基本となるbuttonとresetの説明をします。 ボタンの表面に表示されるラベルはvalueに記載します。ボタンの名称はnameに記載します。 イベント処理のonclickと組み合わせることでボタン押下時の処理を記載することもできます。こちらは次章で説明します。

リセット処理

Buttonタグのtypeにresetを設定すると、リセットが行われます。 リセットすると各タグのデータが初期値(=valueで設定した値)に戻ります。 ただし、このリセットの有効範囲はformタグ内だけですので注意してください。

リセットサンプルコード

<form>
<input type="text" name="textbox1" value="ここの値をコピーします。"><br>
<input type="text" name="textbox2"><br>
<button type="button" onclick="textbox2.value=textbox1.value;">テキストコピー</button>
<button type="reset">リセット</button>
</form>
