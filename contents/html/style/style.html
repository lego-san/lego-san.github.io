HTML Style属性とは

HTML StyleはHTML要素の色やフォント、サイズなどを変更するときに使用できる設定項目(属性)です。

HTML Styleの3つの設定方法

設定方法は3種類あります。タグに直接記載するインライン設定、ヘッダ内に記載するインターナル設定、そして外部ファイルを参照する外部設定。 以下にてそれぞれ紹介します。

[a]インライン設定

インライン設定はタグの中に直接Style情報を記載します。本文中の特定箇所だけ個別に設定できるので便利です。 ただ、使いすぎると保守性が低下するので、インターナル設定や外部設定と併用することをお勧めします。

インライン設定の例

<p style="color: red;">インライン設定の例</p>

[b]インターナル設定

インターナル設定はヘッダの中にStyle情報を記載します。タグごとや、特定のID、Classに対してまとめて設定することができます。 以下の例では全ての<p>要素に対して文字が赤くなるような設定をしています。 ウェブページごとにまとめて設定したい場合に使用してみてください。

インターナル設定の例

<!DOCTYPE html>
<html lang="ja" dir="ltr">
  <head>
    <style>
      p {color: red;}
    </style>
  </head>
  <body>
    <p>インターナル設定の例</p>
  </body>
</html>

[c]外部設定

外部設定は外部のCSSファイルにStyle情報を記載します。 Style情報をウェブページに対してまとめて設定するときに便利です。 ただし、Style情報の変更が参照している全てのファイルに影響ができることに注意してください。

外部設定の例(HTMLファイル)

<!DOCTYPE html>
<html lang="ja" dir="ltr">
  <head>
    <link rel="stylesheet" href="./style.css">
  </head>
  <body>
    <p>外部設定の例</p>
  </body>
</html>

外部ファイルを参照するときは、下記のようにCSSファイルを引き当てる必要があります。

外部ファイルの参照方法

<link rel="stylesheet" href="./style.css">

外部ファイルの記載方法はインターナル設定の記述と同じです。ファイルの拡張子は".css"としてください。

外部設定の例(外部ファイル/style.css)

p {color: red;}

CSS Fonts設定 色

色はcolorで設定できます。

色の設定の例

<head>
  <style>
    p {color: red;}
  </style>
</head>

設定値はred,blue,black,whiteのように文字列で指定する方法や、#FFFなど数値(16進数)で指定する方法などがあります。 色のバリエーションはこちらを参照してください。

感想

文字の装飾は奥が深いです。一番お手軽なのはインライン設定かもしれませんが、 ホームページ等をつくるなら、メンテナンス性を考えて、外部ファイルに記載しておいたほうがいいかもしれません。 ただ、慣れないうちはインラインとインターナルで運用することをお勧めします。
