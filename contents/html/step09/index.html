1x1の表

1x1の表はこのように書きます。


  <table>
    <tr>
      <td>(1,1)</td>
    </tr>
  </table>
(1,1)

1x2の表

1x2の表はこのように書きます。


  <table>
    <tr>
      <td>(1,1)</td>
      <td>(1,2)</td>
    </tr>
  </table>
(1,1) (1,2)

2x2の表

2x2の表はこのように書きます。


  <table>
    <tr>
      <td>(1,1)</td>
      <td>(1,2)</td>
    </tr>
    <tr>
      <td>(2,1)</td>
      <td>(2,2)</td>
    </tr>
  </table>
(1,1) (1,2)
(2,1) (2,2)

表のタイトルを入れてみる

表のタイトルはこのように書きます。


  <table>
    <tr>
      <th>1列目</th>
      <th>2列目</th>
    </tr>
    <tr>
      <td>(1,1)</td>
      <td>(1,2)</td>
    </tr>
    <tr>
      <td>(2,1)</td>
      <td>(2,2)</td>
    </tr>
  </table>
1列目 2列目
(1,1) (1,2)
(2,1) (2,2)
