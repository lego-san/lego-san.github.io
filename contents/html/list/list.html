リストの種類

HTML ListはOrder List、Unordered List、Description Listの3種類あります。それぞれ紹介しています。

Order HTML List

Order Listは番号が振ってあるリストです。手動で番号を振らなくても、自動で付与されるため便利です。デフォルト設定の場合、 番号は数字で表示されます。これはリストのtype属性を変更することでローマ数字表示などに変更することができます。

  2. Unordered List
  3. Description List

Unordered HTML List

Unordered Listは番号が振っていないリストです。項目のマーカーはデフォルト設定では●となっていますが、list-style-typeを設定することで変更することができます。

Description List

Description Listは用語定義等に用いられるリストです。dlタグがディスクリプションリストを示し、dtタグが用語や名称、ddタグがそれぞれの概要を示します。

HTMLサンプル

<dl>
  <dt>Coffee</dt>
  <dd>- black hot drink</dd>
  <dt>Milk</dt>
  <dd>- white cold drink</dd>
</dl>

ディスクリプションリスト出力結果

Coffee
- black hot drink
Milk
- white cold drink

感想

箇条書きの項目数が多い場合や、目次を生成するときにリストは活躍します。使い方を覚えてなてなくても問題はありませんが、 覚えておくと便利です。
