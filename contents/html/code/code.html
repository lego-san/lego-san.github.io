目次

HTML Computer Code Elements

HTMLには下記のような様々なタグが用意されています。

Computer Code Tag
タグ名概要
codeプログラミングコード
kbdキーボード入力
sampコンピュータ出力
var変数
pre整形済みテキスト

codeタグ

codeタグはプログラミングコードに使用されます。コードはmonospace fontで表現されます。空白や改行は表現されないため注意が必要です。

使用例：codeタグ


  <code>
    x = 5;<br>
    y = 6;<br>
    z = x + y;
  </code>

出力結果

x = 5;
y = 6;
z = x + y;

kbdタグ

kbdタグはキーボード入力の表現に使用されます。

使用例：kbdタグ


  <p>Save the document by pressing <kbd>Ctrl + S</kbd></p>

出力結果

Save the document by pressing Ctrl + S

sampタグ

sampタグはコンピュータの出力結果の表現に使用されます。

使用例：sampタグ


  <p>If you input wrong value, the program will return <samp>Error!</samp></p>

出力結果

If you input wrong value, the program will return Error!

varタグ

varタグは変数の表現に使用されます。

使用例：varタグ


  <p>Einstein wrote: <var>E</var> = <var>mc</var><sup>2</sup>. </p>

出力結果

Einstein wrote: E = mc2.

Google code-prettify

Google code-prettifyはGoogleが提供するJavaScriptです。JavaScriptを読み込んでclassを指定すると下記のように表現できます。

ヘッダ定義


  <script type="text/javascript" src="https://cdn.rawgit.com/google/code-prettify/master/loader/run_prettify.js"></script>

使用例：Google code-prettify


  <pre style="margin: 0px;" class="prettyprint lang-js">console.log('Google code-prettify')</pre>

出力結果

console.log('Google code-prettify')

Google code-prettifyを使った行番号表示

classにlinenumsを追加すれば簡単に行番号を入れることができます。ただし、下記のようにstyleを設定する必要があります。

ヘッダ定義


  <style media="screen">
    .prettyprint ol.linenums > li {
      list-style-type: decimal;
    }
  </style>

使用例：Google code-prettify


  <pre style="margin: 0px;" class="prettyprint lang-js">console.log('Google code-prettify')</pre>

出力結果

  x = 5;
  y = 6;
  z = x + y;
  return z;

 window.addEventListener("load", function(event) {
