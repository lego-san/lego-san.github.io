MetaTagについて学んでみる

SEO対策で必須といわれているMeta Tagについて学んでみます。

目次

Meta Tagとは

Metadata is data (information) about data.

Metadataを理解するのが難しいですが、要はデータ情報ということでしょう。 Meta Tagにはサイトのタイトルや著者、サイトタイプをデータとして記述しておくことができます。

Meta Tagの種類(属性)

属性値概要
charsetcharacter set HTMLドキュメントの文字コードを設定します。
UTF-8を設定しておけばOKです。
nameauthor
description
viewport
など		 Metadataの名称を設定できます。
データの名前だと理解してください。中身はcontentで記述します。
contenttext データの内容を記述します。

Meta Charset


  <meta charset="utf-8">

Meta Name Author


  <meta name="author" content="lego-san">

Meta Name Description


  <meta name="description" content="Meta Tagの概要とサンプルについて説明。">

The Open Graph protocol(OGP)とは

FaceBookなどのソーシャルメディアで使用可能なMetadataです。 これを記載していれば概要やサムネイルなどを表示することができます。

OGP Basic Metadata

属性値概要
propertyog:titleページタイトルタイトル
propertyog:typeページタイプ
propertyog:imageページを代表する画像
propertyog:urlページURL 

<html prefix="og: http://ogp.me/ns#">
<head>
  <title>The Rock (1996)</title>
  <meta property="og:title" content="The Rock" />
  <meta property="og:type" content="video.movie" />
  <meta property="og:url" content="http://www.imdb.com/title/tt0117500/" />
  <meta property="og:image" content="http://ia.media-imdb.com/images/rock.jpg" />
...
</head>
...
</html>

OGP Optional Metadata

属性値概要
propertyog:description簡単な概要
propertyog:locale場所。日本の場合はja_JP
propertyog:site_nameサイトネーム 

<meta property="og:audio" content="http://example.com/bond/theme.mp3" />
<meta property="og:description"
  content="Sean Connery found fame and fortune as the
           suave, sophisticated British agent, James Bond." />
<meta property="og:determiner" content="the" />
<meta property="og:locale" content="en_GB" />
<meta property="og:locale:alternate" content="fr_FR" />
<meta property="og:locale:alternate" content="es_ES" />
<meta property="og:site_name" content="IMDb" />
<meta property="og:video" content="http://example.com/bond/trailer.swf" />
