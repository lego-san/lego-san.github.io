marginによる余白

marginの設定とは？

marginは余白です。marginを設定していない文章と文章の間に余白がなくなり窮屈にみえます。 赤色の文章1と青色の文章2の余白を変化させて、marginの余白とは何かをみていきます。

文章1,2共に margin = 0px の場合

文章1 margin = 0px

文章2 margin = 0px

文章2のみ margin = 20px の場合

文章1 margin = 0px

文章2 margin = 20px

文章2のみ margin = 100px の場合

文章1 margin = 0px

文章2 margin = 100px

なんとなく分かりましたか？灰色部分の余白がmarginです。

paddingによる余白

次はpaddingです。

文章1,2共に padding = 0px の場合

文章1 padding = 0px

文章2 padding = 0px

文章2のみ padding = 20px の場合

文章1 padding = 0px

文章2 padding = 20px

文章2のみ padding = 100px の場合

文章1 padding = 0px

文章2 padding = 100px

文章1の赤色領域と文章2の青色領域が接していますが、 paddingを大きくすると文字自体は設定値の分だけ中央に寄っていることがわかります。

margin と paddingの比較

こんな感じです。

文章2 margin = 100px、文章3 padding = 100px の場合

文章1 margin = 0px, padding = 0px

文章2 margin = 100px, padding = 0px

文章3 margin = 0px, padding = 100px
